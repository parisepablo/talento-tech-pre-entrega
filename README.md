# Pre-entrega de proyecto de Automatización QA — Talento Tech

Pre-entrega del curso de **Automatización QA** dado por Talento Tech, cuyo objetivo es desarrollar un conjunto de pruebas automatizadas sobre un sitio web usando Python, Selenium y pytest.

---

## Tecnologías utilizadas

- **Python** — lenguaje principal
- **Selenium** — automatización del navegador (Chrome)
- **pytest** — framework de testing

---

## Estructura del proyecto

```
talento-tech-pre-entrega/
├── tests/          # Casos de prueba automatizados
├── utils/          # Utilidades y helpers
├── conftest.py     # Fixtures de pytest
└── requirements.txt
```

---

## Instrucciones

**1. Clonar el repo**

```bash
git clone https://github.com/parisepablo/talento-tech-pre-entrega.git
cd talento-tech-pre-entrega
```

**2. Instalar dependencias**

```bash
pip install -r requirements.txt
```

**3. Ejecutar las pruebas**

```bash
pytest -v
```
